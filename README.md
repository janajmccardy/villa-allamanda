# Villa Allamanda - Eleuthera, Bahamas

Boutique hotel website built with Vue 3, TypeScript, and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Booking System

This site uses [FreeToBook](https://www.freetobook.com/) for reservations. The booking URL is already configured in the components.

## Deployment

This project is configured for deployment on [Vercel](https://vercel.com). Simply connect your GitHub repository and Vercel will auto-deploy on every push.
